# markdown 基础语法

### 文本断行  
	任何文本紧跟2个空格则自动断行。  
### 链接
	语法：[链接显示文字](链接地址)
	示例代码：[百度一下，你就知道](http://www.baidu.com)
[百度一下，你就知道](http://www.baidu.com)

### 图片链接
	语法：![链接显示文字](链接地址)
	示例代码：![一只猫](http://tomcat.apache.org/res/images/tomcat.png)
![一只猫](http://tomcat.apache.org/res/images/tomcat.png)
